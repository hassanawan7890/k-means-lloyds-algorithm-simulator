# KMeans Simulator

Deployable app folder for the live simulator.

Live demo: https://kmeans-simulator.vercel.app

## Run locally

```powershell
cd C:\Users\Hassan\Downloads\A4\kmeans-app
python -m http.server 4173 --bind 127.0.0.1
```

Open `http://127.0.0.1:4173/`.

## Contents

- `index.html` - app shell and content pages
- `styles.css` - visual design and layout
- `app.js` - simulator logic and interactions
- `input.txt` - sample dataset
- `vercel.json` - deployment config
