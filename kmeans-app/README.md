# K-Means & Lloyd's Algorithm Simulator

Interactive browser-based simulator for k-means clustering and the Lloyd-Forgy update cycle.

Live demo: https://k-means-lloyds-algorithm-simulator.vercel.app

## Run locally

```powershell
cd C:\Users\Hassan\Downloads\A4\kmeans-app
python -m http.server 4173 --bind 127.0.0.1
```

Open `http://127.0.0.1:4173/`.

## Features

- manual point and centroid entry
- click-to-place editing on the simulation plane
- import of the original A4 text format
- random dataset generation with reproducible seeds
- adjustable delay, frame count, epsilon, and max-iteration stopping
- guide, about, and feedback pages in the same static app
